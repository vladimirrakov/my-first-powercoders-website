Scrimba’s HTML and CSS course

July 7, 2023

In April 2023, I completed Scrimba’s HTML and CSS course and created a Bern City Guide Page as a solo project.

The goal of the course was to learn how to build my very first web pages. Scrimba's HTML and CSS program was designed by Kevin Powell, one of the most popular HTML & CSS instructors on the web and covered the following topics:

  • What is HTML and CSS?
  • Terminology and syntax
  • HTML tags
  • File naming and project setup
  • Anchors
  • Attributes
  • strong and emphasis
  • CSS basics
  • Classes and Ids
  • Internal & external CSS
  • Borders
  • The box model
  • Margin & padding
  • Lists
  • Images
  • Centering elements
  • Styling your page
