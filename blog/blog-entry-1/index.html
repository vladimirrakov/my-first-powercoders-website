5-day Coding Challenge

July 7, 2023

In March 2023, I took part in a "5-day Coding Challenge" designed by the Code Institute for the beginner programmers. The objective of this free coding bootcamp was to learn the basics of HTML, CSS & JavaScript and discover if coding is the career path for me.

I have learned the basics of HTML, CSS and JavaScript with this coding challenge and built my first "Tea Recipe Webpage". Thanks to this course I have felt an aptitude for coding and got the initial programming experience.
